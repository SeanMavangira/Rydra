# Rydra – Income, Expense and Trip Tracker

Rydra is a SwiftUI-based mobile application designed for taxi and kombi drivers to track trips, income, and expenses in a simple and structured way.

The app helps drivers understand their daily performance and overall profitability by organizing financial data into clear, easy-to-use views.

## Overview

Rydra focuses on three core areas:

- Trips – Record and manage daily trips  
- Income – Track fares earned  
- Expenses – Log fuel and other operational costs  

All data is presented in a clean interface to give users a clear picture of their financial activity.

## Features

- Add and manage trips  
- Track income from each trip  
- Record expenses such as fuel and maintenance  
- View daily totals and profit  
- Simple and intuitive SwiftUI interface  

## Tech Stack

- Swift  
- SwiftUI  

## Project Structure

- `Views/` – UI built with SwiftUI  
- `Resources/` – Assets and supporting files
